# config-harvesting

| item                    | description                                                                              |
| ----------------------- | ---------------------------------------------------------------------------------------- |
| harvestKey              | `isAutoHarvestingEnabled`がtrueのときに使用する秘密鍵。ひとつだけ指定可能。              |
| isAutoHarvestingEnabled | `true`ならノード起動時に自動的に指定の秘密鍵を使用してハーベスティングを開始する         |
| maxUnlockedAccounts     | **自動ハーベスティングに使用するアカウントを含む**、このノードの最大委任許容アカウント数 |
